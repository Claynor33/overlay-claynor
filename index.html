Configuration — génère tes URL OBS toutes prêtes.

1 · Ton token Streamlabs

Il n'est jamais enregistré dans le dépôt : il n'est ajouté qu'aux URL ci-dessous, chez toi. Streamlabs → Settings → API Settings → API Tokens → Socket API Token.

2 · Tes sources OBS

Widgets (interface reliée — change de jeu et tout se recolore)
Scènes tout-en-un
Pour chaque source Navigateur dans OBS : « URL » = colle l'adresse copiée ; mets la taille indiquée. « Copier » met l'URL dans le presse-papiers ; « Ouvrir » l'affiche dans un onglet pour vérifier.

3 · Ordre des sources dans OBS

  1. celebration tout en haut (plein écran, invisible au repos)
  2. alertes, puis tes widgets (jeu-trophees, soutiens, objectif, camera)
  3. Ta webcam juste sous le widget camera, calée dans le cadre
  4. La capture du jeu tout en bas
Change de jeu via clic droit sur jeu-trophees → Interagir → champ « Jeu » → OK. Toute l'interface se recolore instantanément.
